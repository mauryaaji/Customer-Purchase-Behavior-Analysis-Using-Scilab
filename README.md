# 📊 Customer Purchase Behavior Analysis Using Scilab

## 📌 Description

This project analyzes customer purchase data using Scilab. It reads data from a CSV file and generates insights like product sales, category distribution, monthly trends, and customer segmentation using graphical visualization.

## 🚀 Features

*   CSV data processing
*   Product-wise sales analysis
*   Category distribution (Pie Chart)
*   Monthly sales trend (Line Graph)
*   Top transactions analysis
*   Customer segmentation (High, Medium, Low)
*   Multiple graph visualizations

## 📂 Project Structure

```
Scilab_Project/
│── sales_data.csv
│── scilab sales project.sci
│── README.md
```

## 🛠️ Requirements

*   Scilab installed (recommended version 6.x or above)

## ▶️ How to Run

1.  Download or clone this repository
2.  Open Scilab
3.  Set the project folder as current directory
4.  Run the script:

```
exec('scilab sales project.sci', -1)
```

## 📊 Output

The program generates:

*   Product-wise bar graph
*   Category-wise pie chart
*   Monthly sales trend graph
